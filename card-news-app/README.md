# 카드뉴스 현지화 도구 (로컬 실행형)

영어권 인스타그램 카드뉴스를 한국어 카드뉴스로 바꿔주는 **로컬 실행 프로그램**입니다.
실행 파일을 더블클릭하면 로컬 서버가 켜지고 브라우저에서 작업 화면이 자동으로 열립니다.

## 준비물

- **Node.js** (LTS) — https://nodejs.org 에서 설치 (최초 1회)
- **Claude API 키** — https://console.anthropic.com 에서 발급

## 실행 방법

운영체제에 맞는 파일을 더블클릭하세요.

| OS | 실행 파일 |
|----|-----------|
| Windows | `start.bat` |
| macOS | `start.command` (최초 1회: 우클릭 → 열기) |
| Linux | `start.sh` |

실행하면 브라우저에 `http://127.0.0.1:8787` 이 자동으로 열립니다.
창을 닫으려면 실행된 콘솔 창에서 `Ctrl+C`.

### 먼저 화면만 확인하려면 (데모 모드 · API 키 불필요)

API 키 없이 UI와 동작을 바로 보고 싶으면 **데모 실행 파일**을 더블클릭하세요.
예시(잔 칼망) 번역 결과가 자동으로 채워져, 원본 이미지만 넣으면 카드가 생성됩니다.

| OS | 데모 실행 파일 |
|----|-----------|
| Windows | `demo.bat` |
| macOS | `demo.command` |
| Linux | `demo.sh` |

### API 키 등록 (최초 1회)

두 가지 방법 중 하나:

1. **화면에서 입력** — 페이지 상단 `⚙️ 설정`을 열고 API 키를 붙여넣고 저장.
   (이 폴더의 `config.json`에만 저장되며 외부로 전송되지 않습니다.)
2. **파일로 입력** — `config.example.json`을 `config.json`으로 복사하고 키를 채워 넣기.

## 사용 순서

1. **① 입력**
   - 카드뉴스 이미지(영문) 업로드 / 드래그 / 붙여넣기
   - 게시물 캡션(영문) 붙여넣기
   - 원본 이미지: 역방향 이미지 검색으로 찾은 **URL 입력** 또는 **파일 업로드**
   - `① 분석·번역하기` → Claude가 헤드라인을 읽어 한국어로 번역, 캡션도 번역
2. **② 스타일**
   - 번역된 헤드라인 편집(줄바꿈으로 줄 분리)
   - **폰트 선택**(기본 제공 8종) 또는 **커스텀 폰트(.ttf/.otf/.woff2) 업로드**
   - 글자 크기 / 글자색(흰색·노랑) / 비율(4:5·1:1) / 브랜드 라벨
   - `② 이미지 생성`
3. **③ 결과**
   - 완성 카드 `PNG 다운로드`
   - 번역된 캡션 `복사`

## 설정 값 (`config.json`)

- `apiKey` — Claude API 키
- `model` — 사용할 모델 (기본 `claude-sonnet-5`, 품질 우선이면 `claude-opus-4-8`)

## 환경변수 (선택)

- `PORT` — 시작 포트 (기본 8787, 사용 중이면 자동으로 다음 포트)
- `ANTHROPIC_API_KEY` — config.json 대신 환경변수로 키 지정
- `CARDNEWS_NO_OPEN=1` — 시작 시 브라우저 자동 열기 끄기

## 참고

- 원본 이미지 URL이 프록시로 안 열리면(호스트 차단 등) **파일 업로드** 탭으로 넣으면 됩니다.
- 폰트는 인터넷 연결 시 웹폰트를 사용하며, 커스텀 폰트는 오프라인에서도 동작합니다.
- 이 도구는 원본 이미지의 저작권을 확인해 주지 않습니다. 재게시 시 출처·권리를 직접 확인하세요.
