# Ctrl+C 복사 (Mac용) 크롬 확장 프로그램

Mac 크롬에서 **`Ctrl + C`** 로 복사가 되게 해 주는 확장 프로그램입니다.
붙여넣기는 Mac 기본대로 **`Command + V`** 를 그대로 사용합니다.

## 동작 방식

| 단축키 | 동작 |
|--------|------|
| `Ctrl + C` | 선택한 텍스트 복사 (이 확장이 추가해 주는 기능) |
| `Command + C` | 복사 (Mac 기본, 계속 동작) |
| `Command + V` | 붙여넣기 (Mac 기본) |

- 선택된 텍스트가 있을 때만 복사가 실행됩니다.
- 브라우저 툴바의 확장 아이콘을 눌러 **켜기/끄기** 를 전환할 수 있습니다.

## 설치 방법 (개발자 모드로 불러오기)

크롬 웹 스토어에 올리지 않고 바로 쓰려면 아래처럼 "압축 해제된 확장 프로그램"으로 불러오면 됩니다.

1. 크롬 주소창에 `chrome://extensions` 입력 후 이동
2. 오른쪽 위 **개발자 모드(Developer mode)** 를 켭니다
3. **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 클릭
4. 이 `ctrl-c-copy-extension` 폴더를 선택
5. 툴바에 파란색 **C** 아이콘이 생기면 완료

설치 후에는 어떤 웹페이지에서든 텍스트를 선택하고 `Ctrl + C` 를 누르면 복사됩니다.

## 다른 사람 PC에 배포하기

크롬 웹 스토어에 올리지 않고도 지인·동료 PC에서 쓸 수 있습니다. **폴더를 통째로 압축해 전달** 하면 됩니다.

### 1) 배포용 zip 만들기

이미 만들어 둔 `ctrl-c-copy-extension.zip` 파일을 그대로 보내도 되고, 파일을 수정했다면 아래로 다시 만드세요.

```bash
./build.sh          # ctrl-c-copy-extension.zip 생성
```

> Windows 라면 `manifest.json`, `content.js`, `popup.html`, `popup.js`, `icons` 폴더를 선택해 우클릭 → "압축(zip)" 으로 묶어도 됩니다.

### 2) 받은 사람이 설치하기

1. zip 파일을 원하는 위치에 **압축 해제**
2. 크롬 주소창에 `chrome://extensions` 이동
3. 오른쪽 위 **개발자 모드(Developer mode)** 켜기
4. **압축해제된 확장 프로그램을 로드합니다 (Load unpacked)** 클릭
5. 압축을 푼 폴더 선택 → 파란 **C** 아이콘이 생기면 완료

> 압축을 푼 폴더는 지우거나 옮기지 마세요. 크롬이 그 폴더를 계속 참조합니다.
> (설치 후에도 "개발자 모드 확장 프로그램" 안내 배너가 뜰 수 있지만 정상 동작합니다.)

### 크롬 웹 스토어 정식 등록 (선택)

불특정 다수에게 배포하거나 배너 없이 쓰고 싶다면 [크롬 웹 스토어 개발자 대시보드](https://chrome.google.com/webstore/devconsole)에 등록할 수 있습니다. 최초 개발자 등록비(약 $5)와 심사가 필요하며, 등록 시 위 `build.sh` 로 만든 zip 을 그대로 업로드하면 됩니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `manifest.json` | 확장 프로그램 설정 (Manifest V3) |
| `content.js` | `Ctrl + C` 를 감지해 복사를 실행하는 스크립트 |
| `popup.html` / `popup.js` | 툴바 아이콘 팝업 (켜기/끄기 토글) |
| `icons/` | 확장 아이콘 (16 / 48 / 128px) |
| `build.sh` | 배포용 zip 을 만드는 스크립트 |
| `ctrl-c-copy-extension.zip` | 배포용 압축본 (그대로 전달해 사용) |

## 참고

- Manifest V3 기반이라 별도의 빌드 과정 없이 그대로 동작합니다.
- 브라우저 내부 페이지(`chrome://`, 웹 스토어 등) 일부에서는 보안 정책상 동작하지 않을 수 있습니다.
