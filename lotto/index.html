    로또 6/45 번호 추출 가이드

    로또 6/45란?

    로또 6/45는 1부터 45까지의 숫자 중 6개를 중복 없이 맞히는 복권입니다. 매주 추첨을 통해 당첨 번호 6개와 보너스 번호 1개가 결정됩니다. 6개를 모두 맞히면 1등, 5개 + 보너스 번호를 맞히면 2등이 됩니다. 1등 당첨 확률은 약 814만분의 1입니다.

    번호 추출기 사용법

    본 추출기는 게임 수(1~5게임)를 선택하고 버튼을 누르면 1~45 중 중복 없는 6개 번호와 보너스 번호를 자동으로 생성합니다. 직접 번호를 고민하는 수고를 덜어주며, 여러 게임을 한 번에 뽑아 결과를 복사할 수 있습니다.

    포함·제외 번호 활용

    포함 번호는 생일·기념일 등 평소 마음에 둔 번호를 모든 게임에 반드시 넣고 싶을 때 사용합니다. 제외 번호는 빼고 싶은 숫자를 지정해 추첨에서 제외합니다. 또한 실제 로또처럼 번호 구간별로 색이 적용되어(1~10 노랑, 11~20 파랑, 21~30 빨강, 31~40 회색, 41~45 초록) 한눈에 보기 좋습니다.
