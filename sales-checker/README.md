# 📊 매출 확인 프로그램

377 · 2552 · 2179 세 쇼핑몰(`tmg<번호>.cafe24.com`)의 주문 엑셀을 자동으로 내려받아,
아래 4가지를 한 화면에서 정리해 보여주는 Windows용 프로그램입니다. 웹사이트에 올리지 않고
**실행 파일(또는 배치 파일)로 실행하면 브라우저에 결과 페이지가 열립니다.**

## 무엇을 정리해 주나요

대상 쇼핑몰: **어현준(377) · 이동욱(2552) · 황민호(2179)**

각 쇼핑몰마다:

1. **마지막마켓주문일자** — `마켓주문일자` 열에서 현재 시각에 가장 가까운(가장 최신) 주문일시
2. **해당 월 총 매출** — `더망고주문상태`가 **"반품/교환/취소 진행중"**, **"반품/교환/취소완료"**
   인 건을 제외한 나머지 `총 결제금액합계(원)`의 합계 (`마켓주문일자` 기준 월로 구분)
3. **롯데ON 일자별 매출** — 마켓명이 롯데ON인 건만 일자별 합계 (위 제외 조건 적용)
4. **마켓주문일자별 매출** — 위 제외 조건 적용, 일자별 합계
5. **마켓명별 매출** — 위 제외 조건 적용, 마켓별 합계
6. **상품 카테고리별 매출** — **최근 3일(오늘 포함)** 데이터만, 카테고리별 건수·합계

맨 위에는 **오늘 날짜 매출**(3개 쇼핑몰 합계 및 각 쇼핑몰별)이 크게 표시되고, 그 아래에
이달 합계와 **마지막마켓주문일자**가 함께 보여집니다. 상단에서 **월을 선택**하면 그 달 기준으로
다시 계산합니다(최근 18개월). 기본값은 이번 달입니다.

> 카테고리 집계 기간(기본 3일), 롯데ON 판별 키워드 등은 `sales_checker.py` 상단의
> `CATEGORY_DAYS`, `LOTTEON_KEYS` 에서 바꿀 수 있습니다.

## 사용 방법

### 방법 A. 실행 파일(.exe)로 사용 — 권장
1. 파이썬이 설치된 PC에서 **`exe_만들기.bat`** 을 한 번 더블클릭 → `dist\매출확인.exe` 생성
2. 이후에는 **`매출확인.exe`** 를 더블클릭하면 검은 창이 뜨고 잠시 후 브라우저에 결과가 열립니다.
   - `매출확인.exe` 는 파이썬 없는 다른 PC로 복사해서 사용해도 됩니다.
   - 프로그램을 끄려면 검은 창을 닫으면 됩니다.

### 방법 B. 배치 파일로 바로 사용 (파이썬 설치된 경우)
- **`실행.bat`** 을 더블클릭하세요. 처음 한 번만 필요한 구성요소를 자동 설치한 뒤 브라우저가 열립니다.

> 파이썬이 없다면 https://www.python.org/downloads/ 에서 설치하세요.
> 설치 화면에서 **"Add Python to PATH"** 를 꼭 체크하세요.

## 다운로드가 안 될 때 (수동 파일 넣기)

회사망/보안 프로그램 등으로 자동 다운로드가 막히면, 관리자에서 직접 받은 엑셀을
`data` 폴더에 아래 이름으로 넣어두면 그 파일로 계산합니다.

```
data/377.xls    (또는 377.xlsx)
data/2552.xls
data/2179.xls
```

파일 이름은 **쇼핑몰 번호**로만 지어주세요(확장자는 .xls / .xlsx / .csv 모두 가능).

## 참고

- 다운로드 주소는 선택한 월에 맞춰 자동으로 만들어집니다.
  (예: 7월 = `register_date` 2026-07-01 00:00 ~ 2026-07-31 23:59:59, 한국시간 기준)
- 쇼핑몰 목록이나 표시 이름을 바꾸려면 `sales_checker.py` 상단의 `STORES` 를 수정하세요.
- 이 프로그램은 인터넷의 웹페이지가 아니라, 실행한 PC 안(로컬)에서만 열립니다.
