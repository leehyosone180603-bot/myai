# 📈 주가 예측 프로그램 (Stock Price Predictor)

하나의 국내 주식 종목에 대해 **실시간 주가 흐름과 현재 상황을 파악**하고,
기술적 지표와 **몬테카를로 시뮬레이션**으로 **일자별 예상 주가**를 예측하는
독립 실행형 파이썬 프로그램입니다.

> 예시 종목: **파마리서치 (214450, KOSDAQ)**

웹사이트에 등록되는 도구가 아니라, 터미널에서 직접 실행하는 별도 프로그램입니다.

---

## 주요 기능

- **실시간 시세 수집** — 네이버 금융에서 현재가·등락률·거래량·일봉 히스토리를
  직접 가져옵니다. (파이썬은 브라우저 CORS 제약이 없어 실시간 데이터에 유리)
  - `FinanceDataReader`가 설치돼 있으면 더 풍부한 소스를 자동으로 사용합니다.
  - 네트워크·소스 실패 시 **예시(시드) 데이터로 자동 폴백**하여 항상 결과를 냅니다.
- **현재 상황 진단** — 5·20·60일 이동평균 정배열/역배열, RSI(14) 과매수·과매도,
  연간 변동성 추정으로 상승/하락/중립을 판정합니다.
- **일자별 예측** — 일간 로그수익률의 드리프트·변동성을 추정해
  **몬테카를로 시뮬레이션(기본 4,000회, 오른슈타인-울렌벡 평균회귀)**을 돌리고,
  **주말·공휴일을 제외한 영업일**마다 다음을 계산합니다.
  - 예상 종가(중앙값, 50%)
  - 예상 범위(90% 신뢰구간: 하위 5% ~ 상위 95%)
  - 상승 확률(전일 대비)
- **출력** — 콘솔 표, CSV 저장, 차트 PNG(matplotlib 설치 시).

## 사용법

```bash
# 파마리서치(214450), 10 영업일 예측
python predict.py

# 삼성전자(005930), 20 영업일
python predict.py 005930 --days 20

# 공격적 시나리오 + CSV & 차트 저장
python predict.py 214450 --scenario aggressive --csv result.csv --chart result.png

# 네트워크 없이 예시 데이터로만 실행(오프라인 데모)
python predict.py --no-network
```

### 옵션

| 옵션 | 설명 | 기본값 |
|------|------|--------|
| `code` | 종목코드 6자리 | `214450` |
| `--days N` | 예측 영업일 수 | `10` |
| `--scenario` | `conservative` / `neutral` / `aggressive` | `neutral` |
| `--sims N` | 몬테카를로 시뮬레이션 횟수 | `4000` |
| `--csv PATH` | 예측 결과 CSV 저장 | — |
| `--chart PATH` | 예측 차트 PNG 저장 | — |
| `--manual-price` | 현재가 직접 지정(원) | — |
| `--no-network` | 네트워크 없이 예시 데이터로 실행 | — |

**시나리오 차이**
- `conservative` — 추세 추종 약하게 + 현재가로 강하게 회귀 → 좁고 평탄한 밴드
- `neutral` — 균형
- `aggressive` — 추세 추종 강하게 + 회귀 약하게 → 추세를 넓게 외삽

## 예측 원리 (요약)

1. **데이터** — 최근 약 2년 일봉을 수집하고, 실시간 현재가로 갱신합니다.
2. **지표** — 이동평균(추세), RSI(모멘텀·과열), 일간 수익률의 표준편차(변동성)를 계산합니다.
3. **드리프트 보정** — 과거 평균수익률을 시나리오별로 감쇠하고, 추세 정렬과
   RSI 되돌림을 반영해 하루치 기대 방향(드리프트)을 정합니다. (과도한 외삽 방지: ±2σ 제한)
4. **시뮬레이션** — `x(t+1) = x(t) + κ·(θ − x(t)) + μ + σ·Z` 형태의
   평균회귀 기하 브라운 운동으로 수천 개의 미래 경로를 생성합니다.
   (x = ln(price), θ = ln(현재가), Z ~ 표준정규)
5. **집계** — 각 영업일에서 경로들의 분위수(5/25/50/75/95%)와 상승 비율을 구합니다.

## 데이터 출처

- 네이버 금융 일봉(`siseJson`) 및 실시간 폴링 API (공개 엔드포인트)
- (선택) [FinanceDataReader](https://github.com/FinanceData/FinanceDataReader)

회사 사정·시장 상황에 따라 공개 API 응답 형식이 바뀔 수 있으며, 이 경우
프로그램은 자동으로 예시 데이터로 폴백합니다.

## ⚠️ 투자 유의

본 프로그램은 **과거 통계에 기반한 참고용 시뮬레이션**일 뿐, 투자 자문이나
수익을 보장하는 예측이 아닙니다. 어떤 모델도 미래 주가를 정확히 맞출 수 없으며,
실제 주가는 실적·수급·뉴스 등 예측 불가능한 요인으로 크게 달라질 수 있습니다.
**투자에 대한 모든 판단과 책임은 이용자 본인에게 있습니다.**
