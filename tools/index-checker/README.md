# 색인(검색 노출) 확인 프로그램 — 구글 · 네이버 · 다음

우리 사이트(`calcbox.kr`)의 **모든 페이지가 각 포털 검색에 색인(수집)되어 있는지** 자동으로 확인합니다.
`sitemap.xml`에서 전체 URL을 읽어와, 각 URL을 포털별 `site:` 검색으로 조회한 뒤 **색인 O/X**를 표로 정리합니다.

## ⭐ 가장 쉬운 방법 — 더블클릭 실행

- **Windows**: `색인확인.bat` 더블클릭
- **Mac**: `색인확인.command` 더블클릭 (경고 시 우클릭 → 열기 → 열기)

처음 1회는 필요한 구성요소를 자동 설치(수 분)하고, 이후엔 바로 확인이 실행됩니다.
끝나면 결과(`report.html`)가 자동으로 열립니다.

> Node.js가 없으면 실행 파일이 설치 사이트(https://nodejs.org)를 자동으로 열어줍니다. LTS 버전을 설치한 뒤 다시 더블클릭하세요.

## 결과 보는 법

| 표시 | 의미 |
|------|------|
| ✅ 색인 | 해당 포털 검색에 그 페이지가 노출됨(수집됨) |
| ❌ 미색인 | 아직 검색에 안 나옴(수집 전이거나 제외됨) |
| ⚠️ 확인불가(차단) | 포털이 자동 접근을 일시 제한 — 잠시 후 다시 실행 |

- 각 포털별로 "색인된 페이지 수 / 전체"가 상단에 요약됩니다.
- ❌ 미색인 페이지는 서치콘솔/서치어드바이저에서 **수집(색인) 요청**을 하면 됩니다.

## 설정 바꾸기 (`config.json`)

```json
{
  "sitemap": "https://calcbox.kr/sitemap.xml",  // 확인할 사이트맵 주소
  "domain": "calcbox.kr",                        // 내 사이트
  "engines": ["google", "naver", "daum"],        // 확인할 포털
  "maxUrls": 0                                    // 0 = 전체, 숫자 = 앞에서부터 그만큼만
}
```

## 참고 / 한계

- 포털의 `site:` 검색 결과 기준 **근사 판정**입니다. 새로 발행한 글은 **수집·색인된 뒤에야** ✅로 바뀝니다.
- 특히 **구글**은 자동 검색을 민감하게 차단합니다. `⚠️ 확인불가`가 많으면:
  - 잠시 후 다시 실행하거나,
  - 브라우저 창을 띄워 실행: Windows `set HEADLESS=false && node index-check.js` / Mac `HEADLESS=false node index-check.js`
- 예약발행 대기 중인 글은 아직 사이트맵에 없으므로(발행 시 자동 추가) 확인 대상에 포함되지 않습니다.
