# calcbox.kr 바로가기 실행 파일

더블클릭하면 **calcbox.kr** 사이트가 기본 브라우저로 열립니다.

- **Windows**: `calcbox.kr 바로가기.url` 또는 `사이트열기.bat` 더블클릭
- **Mac**: `사이트열기.command` 더블클릭
  - 처음 열 때 경고가 뜨면: 파일 **우클릭 → 열기 → 열기**

## 바탕화면에 두고 쓰기
`calcbox.kr 바로가기.url` 파일을 **바탕화면으로 복사**해 두면, 언제든 더블클릭으로 사이트를 열 수 있습니다.

## 여는 주소 바꾸기
`사이트열기.bat`(또는 `.command`, `.url`)를 메모장으로 열어 `https://calcbox.kr/` 부분을 원하는 주소로 바꾸면 됩니다.
