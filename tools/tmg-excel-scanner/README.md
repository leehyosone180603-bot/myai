# TMG 매출 엑셀 스캐너

여러 개의 `tmg{번호}` 쇼핑몰(cafe24) 관리자 엑셀 내보내기 주소를 자동으로 훑어서,
**실제 주문 데이터가 들어 있는 순번만** 골라내고, 각 순번의 엑셀로 바로 이동할 수
있는 **버튼**을 만들어 주는 UI 실행 프로그램입니다.

> 두 번째 사진처럼 "주문일자 · 마켓명" 등 실제 값이 들어 있는 순번만 결과에 남고,
> 첫 번째 사진처럼 **열 제목만 있는(데이터 없는) 순번은 자동으로 제외**됩니다.

---

## 왜 실행 프로그램(.exe)인가요?

브라우저(웹페이지)만으로는 `tmg*.cafe24.com` 같은 **다른 도메인**의 엑셀을 내려받아
내용(데이터 유무)을 확인하는 것이 보안정책(CORS) 때문에 막혀 있습니다.
그래서 데이터 확인은 PC에서 도는 실행 프로그램이 담당하고, 확인된 순번은
브라우저에서 바로 열 수 있게 버튼으로 제공합니다.

---

## 주소 규칙

| 순번(tmg 번호) | 사용하는 주소 |
|---|---|
| `1 ~ 3999` | `https://tmg{번호}.cafe24.com/...`  (my **없음**) |
| `4000 ~ 5500` | `https://tmg{번호}.mycafe24.com/...`  (my **포함**) |

프로그램의 **"mycafe24 시작"** 값(기본 4000)으로 경계는 언제든 바꿀 수 있습니다.

---

## 사용 방법

### 방법 A. 파이썬으로 바로 실행 (가장 간단)

1. [python.org](https://www.python.org/downloads/) 에서 파이썬 설치
   (설치 화면에서 **"Add Python to PATH" 체크**)
2. `실행하기.bat` 더블클릭 → 프로그램 창이 열립니다.
   - macOS/리눅스: 터미널에서 `python3 tmg_excel_scanner.py`

### 방법 B. 단독 실행파일(.exe) 만들기 (파이썬 없는 PC 배포용)

1. `exe_빌드하기.bat` 더블클릭
2. 잠시 후 `dist` 폴더에 **`TMG매출엑셀스캐너.exe`** 생성
3. 이 exe는 파이썬이 없는 PC에서도 더블클릭으로 실행됩니다.

## 프로그램 사용 순서

### ① 스캔 탭 — 데이터 있는 순번 찾기
1. **샘플 주소(URL)** 칸에 이번 달 매출 엑셀 주소를 붙여넣기
   (기본값으로 2026년 7월 주소가 이미 들어 있습니다. 다른 달은 새 주소로 교체)
2. **시작 번호 / 끝 번호** 확인 (기본 `1 ~ 5500`)
3. **[스캔 시작]** 클릭 → 진행 막대와 함께 데이터가 확인된 순번이 목록에 하나씩 추가됨
4. 각 줄의 **[엑셀 열기]** 버튼으로 해당 순번 엑셀을 브라우저에서 바로 다운로드
5. **[목록 저장(CSV)]** 로 순번 목록을 파일로 저장

### ② 매출 분석 탭 — 정상주문 매출 집계
스캔이 끝나면 이 탭에서 아래를 자동으로 계산합니다.

1. **정상주문만 집계** — `더망고주문상태` 값에 **취소·반품·교환**이 들어간 주문은 제외
2. **매출 = `결제금액합계(원)` + `결제배송비(원)`**
   (배송비 열이 없는 순번은 결제금액합계만 계산)
3. **순번별 매출을 큰 순서대로 정렬**해서 순위표로 표시
4. **1천만원 이상** 매출 순번은 **상품 카테고리별 매출·건수·비중**까지 정리
5. **[분석결과 저장(CSV)]** / **[리포트 열기(HTML)]** 로 결과를 내보내기
   (리포트에는 순번마다 [엑셀 열기] 버튼 포함)

> 열 이름(주문상태/매출금액/배송비/카테고리)은 실제 엑셀에서 **자동 감지**되며,
> 감지가 어긋나면 상단 드롭다운에서 직접 골라 다시 [매출 분석 실행]을 누르면 됩니다.
> 제외 키워드(취소,반품,교환)와 "큰 매출 기준(기본 1천만원)"도 바꿀 수 있습니다.

## 데이터 유무 판정 방식

- 내려받은 파일이 **xlsx** 이면 → 시트에서 값이 있는 행 수를 셈
  (헤더 1줄만 있으면 데이터 없음 → 제외 / 헤더 + 주문행이 있으면 데이터 있음 → 포함)
- **HTML 표**로 내려오면 → `<tr>` 행 수로 동일하게 판정
- **접속 실패**(존재하지 않는 몰 등)는 조용히 건너뜀

화면 하단 상태줄에 `데이터 확인 / 빈 몰 / 접속실패` 개수가 실시간으로 표시됩니다.

## 설정값 요약

| 항목 | 기본값 | 설명 |
|---|---|---|
| 시작/끝 번호 | 1 / 5500 | 스캔할 tmg 번호 범위 |
| mycafe24 시작 | 4000 | 이 번호부터 `mycafe24` 사용 |
| 동시 접속 수 | 30 | 한 번에 접속하는 개수(빠르게 하려면 늘리기) |
| 응답 대기(초) | 8 | 한 주소당 최대 대기 시간 |

## 파일 구성

| 파일 | 설명 |
|---|---|
| `tmg_excel_scanner.py` | 프로그램 본체 (UI) |
| `tmg_core.py` | 핵심 로직 (주소 생성 · 엑셀 파싱 · 매출 분석) |
| `실행하기.bat` | Windows 실행 (파이썬 필요) |
| `exe_빌드하기.bat` | 단독 `.exe` 생성 (PyInstaller) |
| `README.md` | 사용 설명서 |

> 별도 설치 패키지 없이 파이썬 표준 모듈(tkinter, urllib, zipfile)만으로 동작합니다.
